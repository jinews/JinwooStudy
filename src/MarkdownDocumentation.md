# 1. Headings
Use the # symbol to create headings, with increasing numbers of # for lower-level headings.

This is an H1
=============
This is an H2
-------------

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

- - -

# 2. Bulleted Lists
### You can use  `-` , `*` or `+` to create bulleted lists.

- First item
  - Second item
  - Sub-item

- - - 

# 3. Numbered Lists
### Use numbers followed by a `.` create ordered lists.
1. First item
2. Second item
   1. Sub-item

- - - 
# 4. Text Formatting 
### You can styled text to be `bold`, `italic`, or `bold italic`

- Bold: **Blod** or __Bolid__
- Italic: *Italic* or _Italic_
- Bold Italic: ***Bold Italic*** or ___Bold Italic___

- - - 
# Blockquotes
### Use the > symbol of blockquotes.
> this is a blockquote.
>> Nested blockquote.

- - -

# 6. Code
### There are two ways to include code: inline code and blocks.
- Inline Code: Warp text with backticks. Example: `code`
- Code Block: Use triple backticks and optionally specify the language for syntax highlighting.
### Example
```
public static void main ( String args[] ) {
    System.out.println(" Sample Code " ) ;
}
```

# 7. Horizontal Lines
### To create a horizontal, use three or more `-`,`*` or `_` symbols
* * *
- - -
_ _ _

# 8.Links
### To add a hyperlink, wrap the text in square brackets, followed by the URL in parentheses
[NAVER](www.naver.com)
